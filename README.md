# Получение статистика с HH и SJ

Скрипт выводит статистику по количеству вакансий, средней зарплате по 10 популярным ЯП с HeadHunter и SuperJob

# Установка и запуск
1. Склонировать проект
```
git clone https://github.com/cedradeff/dev-hh-api
```
2. Установить зависимости из requirements.txt
```
pip install -r requirements.txt
```
3. Вставить в файл .env значения переменной окружения SJ_SECRET_KEY (нужно для работы с API SuperJob).
4. Запустить скрипт
```
python main.py
```
